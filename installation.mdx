---
title: "Install LSFG-Android on your device"
description: "Learn how to download and sideload the LSFG-Android APK on your Android device. Not on Google Play — install via file manager or ADB."
---

LSFG-Android is distributed exclusively as a sideloaded APK and is not available on Google Play. The combination of `SYSTEM_ALERT_WINDOW` (required for the overlay), screen capture via `MediaProjection`, and `AccessibilityService` (required for touch passthrough) violates Google Play policy — this is a fundamental architectural constraint, not an oversight. You must install the APK manually from GitHub Releases.

<Warning>
  LSFG-Android will never appear on Google Play or any other commercial app store. The app's own license also prohibits commercial distribution. Only install APKs from the official [GitHub Releases](https://github.com/FrankBarretta/LSFG-Android/releases) page.
</Warning>

## Installation

<Steps>
  <Step title="Download the APK">
    Go to the [GitHub Releases](https://github.com/FrankBarretta/LSFG-Android/releases) page and download the latest `app-debug.apk`.
  </Step>
  <Step title="Allow installation from unknown sources">
    Android blocks app installs from outside Google Play by default. You need to grant permission either for your file manager app or for ADB, depending on your install method.

    - **Android 8.0 and newer:** Go to **Settings → Apps → Special app access → Install unknown apps**, then enable the toggle for the app you will use to open the APK (your file manager or a browser).
    - **Older versions:** Go to **Settings → Security** and enable **Unknown sources**.
  </Step>
  <Step title="Install the APK">
    Choose the method that suits your setup.

    <Tabs>
      <Tab title="File manager">
        1. Transfer the downloaded `app-debug.apk` to your Android device (via USB, cloud storage, or direct download on the device).
        2. Open your file manager app and navigate to the APK file.
        3. Tap the file. Android will prompt you to confirm the installation.
        4. Tap **Install** and wait for the process to complete.
        5. Tap **Open** to launch the app, or find **LSFG-Android** in your app drawer.
      </Tab>
      <Tab title="ADB">
        If you have Android Debug Bridge set up on your computer, you can install the APK over a USB connection without a file manager.

        1. Enable **USB debugging** on your device under **Settings → Developer options**.
        2. Connect your device to your computer via USB and accept the debugging prompt on the device.
        3. Run the following command from the directory containing the APK:

        ```bash
        adb install app-debug.apk
        ```

        4. Wait for `Success` to appear in the terminal output.
      </Tab>
    </Tabs>
  </Step>
  <Step title="Complete the first-launch setup">
    The first time you open LSFG-Android, a tutorial walks you through the required Accessibility setup: enabling the touch-passthrough service and unblocking the Restricted Settings that Android 13 and newer apply to sideloaded apps. See the [First launch](/setup/first-launch) guide for the full walkthrough.
  </Step>
</Steps>
